# Infrastructure: Conexión Física y Orbital

Sistemas de sensores, energía renovable y satélites para la resiliencia del ecosistema.

## Componentes

- **atacama-iot/:** Red de sensores en Atacama para monitoreo de Litio y energía
- **lunar-link/:** Protocolos satelitales para backup orbital e inmutabilidad

## Propósito

Garantizar que el ecosistema tenga validación de activos físicos reales y redundancia de datos a nivel orbital.
